## Installation

1) install the code from my git repo  

2) run npm install 

3) change .env.example  to  env  

4)  fill out the info  in the new .env 

5) run mysql -u root -p  

6)  run both source db/schema.sql and db/seeds.sql 

7) run node index.js and enjoy
